# @FIAP/sidebar

## Descrição

Microfrontend responsável pela navegação lateral da aplicação, fornecendo acesso rápido às diferentes seções do sistema.

## Tecnologias

- **Framework**: React.
- **Linguagem**: TypeScript.
- **Arquitetura**: Single-SPA.
- **Design System**: [`@valoro/ui`](https://www.npmjs.com/package/@valoro/ui).
- **Build Tool**: Webpack.

## Pré-requisitos

- **Node.js**: Versão LTS (v20+ recomendada).
- **Gerenciador de Pacotes**: pnpm.

## Como Rodar

1. Instale as dependências:

   ```bash
   pnpm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   pnpm start
   ```

## Funcionalidades

- **Navegação**: Links para Home, Analytics, Configurações, etc.
- **Responsividade**: Adaptação para dispositivos móveis (drawer/menu).
